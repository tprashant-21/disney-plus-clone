<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running, follow these steps.

### Installation

1. Clone the repo
```sh
git clone https://github.com/tprashant-21/disney-plus-clone.git
```

2. Install required npm packages   

```sh
npm install
```
     
3 Start the Dev server   

```sh
npm run dev
```
